## 專案介紹 (Program Introduction)

這是一個專為輔仁大學學生設計的**「畢業學分查核與選課推薦系統」**。
系統結合了 Web 網路爬蟲與 Python GUI (CustomTkinter) 技術，自動化幫學生計算畢業門檻，並給予最精準的選課建議與視覺化分析。

### 核心功能列表

1. **自動化個人成績擷取 (`test1.py`)**  
   使用 Selenium 模擬瀏覽器行為，登入「輔大 SIS 學生資訊系統」，將歷年修課的所有課程名稱、學分數與最終成績自動匯入至本地 MySQL 資料庫中。(支援讀取 `account.txt` 自動登入)

2. **畢業學分檢核表爬取 (`test2.py`)**  
   針對「輔大學習輔導與預警系統」的畢業學分檢核頁籤進行爬蟲，精準辨識出哪些校定、院系必修**「已經完成」**，哪些**「尚未修課」**，並同步紀錄進資料庫，做為推薦系統的最高權重依據。

3. **通識課程類別自動化分類 (`get_school_info/scrape_general_edu.py`)**
   全自動化爬取「輔大課程大綱查詢系統」，自動翻頁並深入各課程大綱內抓取「開課單位」，將全校通識課程分類精準存入資料庫，支援上下學期分類。

4. **現代化智慧桌面應用程式 (`test_project/gui.py`)**  
   - **智慧勾選與防呆：** 系統自動比對個人歷年成績與學校課表，當「課名」與「學分」皆吻合時，系統會自動為您打勾，並自動將這些過關課程**置頂顯示**。針對重複修課，系統會自動採計「最高分」的那次成績，並防呆避免同名課程學分數重複計算。
   - **動態選課推薦：** 系統會計算您的「總學分」、「必修」與「選修」落差，優先推薦您尚未通過的必修課程。
   - **視覺化數據與匯出：** 提供精美的學分圓餅圖分析，並支援一鍵匯出為 PDF 報表以供留存與列印。

## 專案開發流程 (Agile & Scrum)

### Epic (史詩)
**建立一個全自動化的輔大畢業學分查核與選課推薦桌面應用程式**
為幫助輔大學生省去手動比對學分與查找課程的時間，系統需自動抓取歷年成績與畢業門檻，提供視覺化的學分分析與智慧化的選課推薦，讓學生能輕鬆規劃畢業藍圖。

### User Story (使用者故事)
1. **作為一個輔大學生**，我希望系統能自動登入 SIS 抓取我的歷年成績並採計最高分，**以便**我不必手動輸入。
2. **作為一個輔大學生**，我希望系統能自動比對我的畢業門檻與通識分類，**以便**我能清楚知道還有哪些學分尚未完成。
3. **作為一個面臨選課的學生**，我希望系統能根據我缺少的學分推薦合適的課程，**以便**我能高效率地選課。
4. **作為一個使用者**，我希望系統擁有現代化且流暢的桌面介面，並能在清單上將已及格課程置頂顯示，**以便**我能有良好的操作體驗。

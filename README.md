## 專案介紹 (Program Introduction)

這是一個專為輔仁大學學生設計的**「畢業學分查核與選課推薦系統」**。
系統結合了 Web 網路爬蟲與 Python GUI (CustomTkinter) 技術，自動化幫學生計算畢業門檻，並給予最精準的選課建議與視覺化分析。

### 核心功能列表

1. **自動化個人成績擷取 (`test1.py`)**  
   使用 Selenium 模擬瀏覽器行為，登入「輔大 SIS 學生資訊系統」，將歷年修課的所有課程名稱、學分數與最終成績自動匯入至本地 MySQL 資料庫中。(支援讀取 `account.txt` 自動登入)

2. **畢業學分檢核表爬取 (`test2.py`)**  
   針對「輔大學習輔導與預警系統」的畢業學分檢核頁籤進行爬蟲，精準辨識出哪些校定、院系必修**「已經完成」**，哪些**「尚未修課」**，並同步紀錄進資料庫，做為推薦系統的最高權重依據。*(此腳本採用 `undetected-chromedriver` 以繞過 Cloudflare 機器人驗證機制)*

3. **現代化桌面應用程式介面 (`test_project/gui.py`)**  
   - 提供基於 CustomTkinter 的深色模式、高質感的現代化桌面應用程式介面，支援流暢的資料渲染與無限捲動體驗。
   - 進入介面後，系統會自動比對資料庫，幫助您檢視已修課程。介面提供過濾與搜尋功能，可依據課程類別、學期與星期快速找到目標課程。

4. **動態選課推薦與視覺化分析 (`test_project/gui.py`)**  
   - 系統會計算您的「總學分」、「必修」與「選修」的落差，並動態計算通識與各類學分門檻。
   - **優先推薦檢核表缺口：** 系統會優先分析 `test2.py` 找到的「尚未修課」必修項目，並推薦能滿足此缺口的課程。
   - **視覺化分析與匯出報表：** 結合 Matplotlib 產生圓餅圖，直觀展示各類學分（必修、選修、通識等）的完成度；並提供匯出 PDF 報告功能，方便留存與列印。

### 執行方式 (自動化流程)

為求方便，專案內準備了快速執行腳本，執行後將自動依序呼叫兩個爬蟲腳本，最後再打開網頁系統：
- **Mac/Linux 用戶:** `./run.sh`  (或 `test_project/run.sh`)
- **啟動 GUI 系統:** `python test_project/gui.py`

## 專案開發流程 (Agile & Scrum)

### Epic (史詩)
**建立一個全自動化的輔大畢業學分查核與選課推薦桌面應用程式**
為了幫助輔大學生省去手動比對學分與查找課程的時間，系統需自動抓取歷年成績與畢業門檻，並提供視覺化的學分分析與智慧化的選課推薦，讓學生能輕鬆規劃畢業藍圖。

### User Story (使用者故事)
1. **作為一個輔大學生**，我希望系統能自動登入 SIS 抓取我的歷年成績，**以便**我不必手動輸入。
2. **作為一個輔大學生**，我希望系統能自動比對我的畢業門檻，**以便**我能清楚知道還有哪些必修與通識尚未完成。
3. **作為一個面臨選課的學生**，我希望系統能根據我缺少的學分與必修推薦合適的課程，**以便**我能高效率地選課。
4. **作為一個需要規劃學習進度的學生**，我希望能在系統內看到圖表分析與匯出 PDF 報表，**以便**我可以追蹤自己的畢業進度。
5. **作為一個使用者**，我希望系統擁有現代化且流暢的桌面介面，**以便**我在操作過濾與搜尋功能時能有良好的體驗。

**目標：** 建立核心爬蟲機制與資料庫架構，完成資料自動化擷取。
* [完成] 建立 MySQL 資料庫與 Table Schema 設計。
* [完成] 開發 `test1.py`：使用 Selenium 自動登入並爬取歷年成績與學分。
* [完成] 開發 `test2.py`：突破 Cloudflare 驗證，爬取畢業檢核表，精準辨識已修與未修項目。
* [完成] 初步的 PHP 網頁版介面 (舊版)：能顯示已修與未修課程清單與過濾。

### Sprint 2 (第二衝刺期) - 桌面化、推薦演算法與視覺化
**目標：** 將系統全面升級為 Python GUI，實作智慧推薦演算法，並加入報表匯出功能。

# 輔大畢業學分查核與選課推薦系統 - 專案報告講稿

## 1. 開場與專案動機 (約 1-2 分鐘)

**[投影片：標題頁 - 輔大畢業學分查核與選課推薦系統]**

各位評審、老師、同學大家好。今天我們要展示的專案是「輔大畢業學分查核與選課推薦系統」。

**[投影片：痛點與動機]**

相信身為輔大的學生，大家每到選課期間都有過這樣的困擾：我們必須登入好幾個不同的系統（像是 SIS 學生資訊系統、學習輔導系統），自己拿著紙筆，一筆一筆核對哪些必修修過了、哪些還沒修。這不僅耗時，還很容易因為算錯而有延畢的風險。

為了解決這個痛點，我們開發了這套系統。它的目標是：讓系統自動幫你抓成績、比對畢業門檻，並直接推薦你這學期該修什麼課。

---

## 2. 核心功能展示 (約 3-4 分鐘)

**[投影片 / 實機展示：自動化成績與門檻擷取]**

首先，我們來看系統的第一個亮點：「自動化資料擷取」。
我們的系統內建了基於 Python Selenium 開發的網路爬蟲（`test1.py` 與 `test2.py`）。只要執行腳本，它就會自動登入 SIS 系統抓取你歷年的修課成績，接著再突破 Cloudflare 驗證，去抓取你的「畢業學分檢核表」。系統會精準掌握你已經修過什麼、還缺什麼必修，並統整到本地資料庫中。

**[投影片 / 實機展示：網頁端學分計算介面]**

資料抓取完畢後，我們進入到 PHP 開發的網頁系統 (`index.php`)。
在這裡，系統會自動比對資料庫，幫你把所有已經及格或抵免的課程「自動打勾」。網頁也提供了過濾系統，你可以依照學期或星期來搜尋課程。

**[投影片 / 實機展示：動態選課推薦]**

接著是系統的核心：「智慧選課推薦 (`process.php`)」。
當你提交查核後，系統會計算你的總學分、必修與選修的落差。最重要的是，它會優先分析出你「尚未修課」的必修項目，並在資料庫中推薦能滿足這些缺口的課程，直到被推薦的學分總數剛好填滿你的畢業門檻為止。

---

## 3. 開發技術架構 (約 1-2 分鐘)

**[投影片：系統架構圖]**

在技術選型上，我們採用了混合式的架構：
*   **爬蟲層**：使用 Python (Selenium, undetected-chromedriver) 模擬人類行為進行自動化登入與抓取。
*   **資料層**：將爬取到的資料清洗後存入 MySQL 資料庫。
*   **前後端層**：使用 PHP 處理邏輯運算與推薦演算法，並透過 Web 介面呈現給使用者。

---

## 4. 結語與未來展望 (約 1 分鐘)

**[投影片：未來展望與結語]**

總結來說，這個系統將繁瑣的人工查核轉變為自動化的智慧體驗。然而，我們的專案還在持續進化中。

**未來我們計畫要完成的項目包含：**
1. 將目前的 PHP 網頁版全面改寫為 **Python GUI (桌面應用程式)**，提升使用流暢度。
2. 加上「分析圖表」和「匯出結果成 PDF」的功能，讓學生能直觀地看到進度並留存報告。
3. 支援更複雜的畢業條件，例如：不同系所的畢業門檻、通識的分類過濾、擋修機制，甚至是雙主修的計算。
4. 最後，將系統包裝成完整的 `.app` 或 `.exe` 執行檔，讓同學可以更方便地安裝使用。

我們的報告到此結束，謝謝大家！

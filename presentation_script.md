# 輔大畢業學分查核與選課推薦系統 - 專案報告講稿

## 1. 開場與專案動機 (約 1-2 分鐘)

**[投影片：標題頁 - 輔大畢業學分查核與選課推薦系統]**

各位評審、老師、同學大家好。今天我要向大家展示的專案是「輔大畢業學分查核與選課推薦系統」。

**[投影片：痛點與動機]**

相信身為輔大的學生，大家都有過這樣的困擾：每到選課期間，總是得手動登入好幾個不同的系統，像是 SIS 學生資訊系統看成績、學習輔導系統看畢業門檻。我們必須自己拿著紙筆，一筆一筆核對哪些必修修過了、通識還差幾學分，不僅耗時，還非常容易算錯，甚至導致延畢的風險。

為了解決這個痛點，我開發了這套全自動化的桌面應用程式。它的目標很簡單：讓學生一鍵登入，系統就能自動幫你算好還缺什麼學分，並直接推薦你這學期該修什麼課。

---

## 2. 核心功能展示 (約 3-4 分鐘)

**[投影片 / 實機展示：自動化資料擷取]**

首先，我們來看系統的第一個亮點：「自動化資料擷取」。
傳統上，學生必須手動輸入成績。但我們的系統內建了強大的網路爬蟲（基於 Selenium 開發）。只要在系統中設定好帳號密碼，它就會在背景自動登入 SIS 系統抓取你歷年的修課成績，接著再突破 Cloudflare 驗證，去抓取你的「畢業學分檢核表」。
這代表系統能夠精準掌握你已經修過什麼、以及「校定」或「院系」必修還缺什麼，並將這些資料統整到本地資料庫中。

**[投影片 / 實機展示：現代化桌面介面]**

資料抓取完畢後，進入到我們的 Python GUI 桌面介面。
我們揚棄了傳統陽春的網頁，採用了 CustomTkinter 打造出支援深色模式、具有高質感的現代化介面。在這裡，你可以清楚看到所有修過的課程。我們也提供了強大的過濾與搜尋功能，你可以根據「學期」、「星期」來快速篩選你想找的課。

**[投影片 / 實機展示：動態選課推薦]**

接下來是系統的核心：「智慧選課推薦」。
系統會自動比對你的修課紀錄與畢業門檻，計算出你目前的學分落差。最聰明的地方在於，它會「優先抓出你還沒修的必修課」，並在最新的開課清單中，推薦能填補這些學分缺口的課程給你。系統會不斷幫你湊齊學分，直到滿足你的畢業門檻為止。

**[投影片 / 實機展示：視覺化分析與匯出]**

為了讓學生對進度更有感，系統還整合了 Matplotlib 圓餅圖。你可以直觀地看到自己必修、選修、通識的完成度百分比。最後，我們還提供了「匯出 PDF 報告」的功能，讓你把這份完整的學分健檢報告印下來或存檔，方便隨時追蹤進度。

## 3. 開發流程與架構 (約 1-2 分鐘)

**[投影片：敏捷開發 Scrum 流程]**

在開發過程中，我導入了敏捷開發 (Agile) 的概念來推進專案：
*   **在 Sprint 1 (第一衝刺期) 中：** 我專注於打好底層基礎。完成了 MySQL 資料庫建置，以及最困難的自動化爬蟲程式，確保系統能穩定、正確地抓到學生的成績和必修缺口。
*   **在 Sprint 2 (第二衝刺期) 中：** 我著手進行「桌面化與智能化」。將初期的網頁介面重構成流暢的 Python GUI，實作了核心的推薦演算法，並加入了數據視覺化及 PDF 匯出功能，大幅提升了使用者體驗。

---

## 4. 結語與未來展望 (約 1 分鐘)

**[投影片：未來展望與結語]**

總結來說，這個系統將原本繁瑣、易出錯的人工查核過程，轉變為全自動、具備視覺化分析的智慧體驗，真正實現了「科技來自於人性，用程式解決生活痛點」。

未來，我希望能進一步擴充這個系統，例如：支援「雙主修」或「輔系」的複雜學分計算、加入更細緻的通識分類過濾，甚至是將它打包成完整的 `.exe` 或 `.app` 安裝檔，讓更多不會寫程式的輔大同學也能輕鬆使用。

我的報告到此結束，感謝各位的聆聽！如果對系統架構或爬蟲細節有興趣，歡迎提問。
